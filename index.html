Welcome

Live Coding is an art form in electronic music and video performance.

The Live Coder creates and manipulates source code live in front of an audience. He combines code, music and visuals into a synaestetic performance.

Normally, the source code is hidden from the user of a program and will never be recognized by him.

All what is left are the bits in the machine.
